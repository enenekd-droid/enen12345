# Learning Portal 개인정보처리방침

시행일: 2026년 6월 27일

Learning Portal(이하 "본 서비스")은 초등학생의 학습 활동을 지원하기 위한 교육용 웹 애플리케이션입니다. 본 서비스는 개인정보 보호법 등 관련 법령을 준수하며, 이용자의 개인정보를 안전하게 보호하기 위해 다음과 같이 개인정보처리방침을 공개합니다.

운영 기관: 서울원묵초등학교  
담당교사명: 김정호  
연락처: 02-3421-2100

## 1. 개인정보의 처리 목적

본 서비스는 다음 목적을 위해 필요한 범위에서만 개인정보를 처리할 수 있습니다.

- 학습 앱 목록, 추천 학습, 최근 학습 등 교육 활동 지원
- 학습 활동 진행 상황 확인 및 수업 운영 보조
- 서비스 오류 확인, 기능 개선, 이용 환경 안정화
- 이용 문의 및 권리 행사 요청 처리

현재 버전의 본 서비스는 로그인, 회원가입, 서버 저장 기능을 제공하지 않는 정적 웹앱입니다. 따라서 별도의 이름, 연락처, 주소, 주민등록번호 등 민감하거나 직접 식별 가능한 개인정보를 입력받지 않습니다.

## 2. 처리하는 개인정보 항목

현재 버전에서 본 서비스가 직접 수집하는 개인정보 항목은 없습니다.

다만, 향후 로그인, 학습 기록 저장, 교사용 관리 기능 등을 추가하는 경우 다음과 같은 항목을 최소 범위에서 처리할 수 있습니다.

| 구분 | 항목 | 처리 목적 |
| --- | --- | --- |
| 학생 계정 정보 | 이름 또는 별명, 학년, 반, 번호, 아이디 | 학습자 구분 및 수업 운영 |
| 학습 기록 | 학습 앱 이용 내역, 점수, 완료 여부, 학습 시간 | 학습 현황 확인 및 피드백 |
| 보호자 동의 정보 | 만 14세 미만 아동의 법정대리인 동의 여부 | 아동 개인정보 처리 적법성 확인 |

본 서비스는 주민등록번호, 주소, 개인 전화번호, 민감정보를 수집하지 않는 것을 원칙으로 합니다.

## 3. 개인정보의 처리 및 보유 기간

현재 버전은 서버에 개인정보를 저장하지 않습니다.

향후 학습 기록 저장 기능을 운영하는 경우 개인정보는 수집·이용 목적 달성 시 또는 해당 학년도 종료 후 지체 없이 파기하며, 학교 운영 방침이나 관련 법령에 따라 보관이 필요한 경우에는 그 기간 동안 안전하게 보관합니다.

예시 보유 기간:

- 학생 계정 정보: 해당 학년도 종료 시까지
- 학습 기록: 해당 학년도 종료 후 2개월 이내 삭제
- 문의 기록: 문의 처리 완료 후 3개월 이내 삭제

## 4. 만 14세 미만 아동의 개인정보 처리

본 서비스는 초등학생을 주요 이용 대상으로 하므로, 만 14세 미만 아동의 개인정보를 처리해야 하는 기능이 추가되는 경우 법정대리인 또는 학교의 적법한 동의 절차를 거칩니다.

동의가 필요한 기능이 제공될 때에는 수집 항목, 이용 목적, 보유 기간, 동의 거부 시 제한되는 서비스 내용을 알기 쉽게 안내합니다.

## 5. 개인정보의 제3자 제공

본 서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다.

다만, 다음의 경우에는 예외로 할 수 있습니다.

- 이용자 또는 법정대리인이 사전에 동의한 경우
- 법령에 특별한 규정이 있거나 관계 기관의 적법한 요청이 있는 경우
- 이용자의 생명, 신체, 재산 보호를 위해 긴급히 필요한 경우

## 6. 개인정보 처리 위탁

현재 버전의 본 서비스는 개인정보 처리를 외부 업체에 위탁하지 않습니다.

다만, 본 서비스는 GitHub 및 Vercel 등 외부 플랫폼을 통해 정적 웹사이트 형태로 배포될 수 있습니다. 이 경우 해당 플랫폼은 서비스 제공 과정에서 접속 로그, IP 주소, 브라우저 정보 등 기술적 로그를 자체 정책에 따라 처리할 수 있습니다.

## 7. 개인정보의 파기 절차 및 방법

개인정보 보유 기간이 지나거나 처리 목적이 달성된 경우 지체 없이 해당 개인정보를 파기합니다.

- 전자적 파일: 복구할 수 없는 방법으로 영구 삭제
- 종이 문서: 분쇄 또는 소각
- 계정 및 학습 기록: 데이터베이스 또는 저장소에서 삭제

현재 버전은 서버 저장 기능이 없으므로 별도 파기 대상 개인정보를 보유하지 않습니다.

본 서비스는 개인정보 보호를 위해 다음과 같은 조치를 적용하거나 적용할 예정입니다.

- 개인정보 수집 최소화
- HTTPS 기반의 안전한 접속 환경 사용
- 관리자 접근 권한 최소화
- 비밀번호 저장 기능 도입 시 단방향 암호화 적용
- 불필요한 개인정보 입력란 배제
- 개인정보 처리 담당자 지정 및 관리

현재 버전에서는 개인정보를 서버에 저장하지 않으므로 삭제 요청 대상 정보가 없을 수 있습니다. 향후 회원가입 또는 학습 기록 저장 기능이 추가되는 경우, 서비스 내 기능 또는 담당자 연락을 통해 권리를 행사할 수 있도록 안내합니다.

## 10. 쿠키 및 로컬 저장소 이용

현재 버전의 본 서비스는 필수적인 서비스 제공을 위해 서버 쿠키를 사용하지 않습니다.

향후 사용 편의를 위해 브라우저의 로컬 저장소를 사용할 수 있습니다. 예를 들어 윤리 가이드 확인 여부, 즐겨찾기, 최근 선택 항목 등을 이용자의 기기에만 저장할 수 있습니다. 이 정보는 서버로 전송되지 않으며, 이용자가 브라우저 데이터를 삭제하면 함께 삭제됩니다.

현재 구현 상태:

- 윤리 핵심가이드 확인 여부: 새로고침 시 다시 확인
- 즐겨찾기: 브라우저 세션 중 화면 동작용으로만 사용
