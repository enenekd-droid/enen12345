# 초등 학습 웹포털 (Learning Portal)

## 1. 프로젝트 개요

### 프로젝트명

Learning Portal (가칭)

### 목적

학생들이 다양한 학습용 웹앱을 하나의 포털에서 쉽게 이용할 수 있도록
한다.

현재는 초등학교 5학년 국어, 수학, 사회, 과학 개념어 학습 앱을 중심으로
시작하며, 향후 영어, AI, 메이커교육, 디지털아트 등 다양한 웹앱을
지속적으로 추가할 수 있는 플랫폼을 구축한다.

------------------------------------------------------------------------

# 2. 프로젝트 목표

-   학습용 웹앱을 한 곳에서 관리
-   학생이 원하는 앱을 쉽게 찾을 수 있도록 제공
-   교과별 분류
-   학습 진행률 확인
-   최근 학습 기록 제공
-   테스트 및 학습 결과 저장
-   모바일 및 PC 모두 사용 가능

------------------------------------------------------------------------

# 3. 타겟 사용자

### Primary

-   초등학생 (3\~6학년)

### Secondary

-   교사
-   학부모

------------------------------------------------------------------------

# 4. 핵심 기능

## 4.1 홈

첫 화면에서 추천 학습, 교과별 메뉴, 최근 학습, 오늘의 학습, 학습 현황을
확인할 수 있다.

## 4.2 교과별 분류

-   국어
-   수학
-   사회
-   과학
-   AI
-   영어
-   기타

## 4.3 검색

키워드(예: 분수, 독도, 광합성, 관용표현)로 관련 앱을 즉시 검색한다.

## 4.4 앱 카드

각 앱은 카드 형태로 표시한다.

표시 항목 - 아이콘 - 앱 이름 - 설명 - 학년 - 교과 - 시작하기 버튼

## 4.5 추천 학습

첫 화면 상단에 추천 앱 표시

## 4.6 최근 학습

최근 실행한 앱과 점수, 날짜 표시

## 4.7 학습 현황

-   완료한 앱 수
-   평균 점수
-   오늘 학습 시간
-   이번 주 학습 수

## 4.8 즐겨찾기

자주 사용하는 앱 저장

## 4.9 반응형 디자인

-   PC
-   태블릿
-   모바일

-   Minimal
-   Friendly

# 7. UX 원칙

-   학생은 3번 이하 클릭으로 원하는 앱 실행
-   검색창은 항상 상단
-   앱은 카드형 UI
-   버튼은 크게 제작
-   모바일에서도 동일한 UX 제공

------------------------------------------------------------------------

# 8. 기술 스택

Frontend - React - Next.js

UI - TailwindCSS - shadcn/ui

Animation - Framer Motion

Backend - Firebase

Hosting - Vercel

Database - Firestore

Storage - Firebase Storage

Authentication - Google Login (선택)
